
# Major Network Effects

| Type of Network Effect | Short Definition | Typical Example |
|-----------------------|------------------|-----------------|
| Direct (same-side) network effects | The value of the product or service increases as more users of the same type join the network. | Messaging or social networking applications. |
| Indirect (cross-side) network effects | The value for one group of users increases as another user group grows. | Marketplaces where more sellers attract more buyers and vice versa. |
| Data network effects | More users generate more data, improving algorithms, predictions, or personalization, which attracts more users. | AI-based platforms and digital services. |
| Learning network effects | System performance improves over time through accumulated usage, feedback, and experience. | Recommendation systems that improve with continued use. |
| Local network effects | Value depends on the density of users within a specific subgroup, location, or community. | Ride-sharing or delivery platforms in a specific city. |
| Two-sided / multi-sided network effects | Platforms gain value by enabling interactions between two or more distinct user groups. | Payment platforms connecting consumers and merchants. |
| Negative (congestion) network effects | Excessive users reduce value due to overcrowding, competition, or reduced service quality. | Overcrowded platforms with limited resources. |
